# Intro_Lab

Sitio web educativo interactivo para estudiantes de los primeros semestres de ingeniería. Explica conceptos de corriente directa (CD) y alterna (CA).  
Incluye teoría, ejercicios y simuladores interactivos.

## Estructura
- Teoría de Corriente Directa y Alterna
- Ejercicios tipo quiz
- Recursos multimedia y simuladores
